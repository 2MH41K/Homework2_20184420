+ 리눅스 명령어에서...

  > **top 명렁어 사용법**
  > ** top [옵션] **
  > top 명령어는 리눅스 시스템이 실행되고 난 후부터 지금까지의 시간과 시스템에 로그인된 사용자 수 그리고 시스템 부하율을 표시하는 명령어인 uptime의 결과를 가장 상단에 표시해주며, 그 아래로 프로세스 수치, CPU 정보, 메모리 정보를 보여주고, 각 프로세스 별 부하율과 상태를 표시해줍니다. 
  >**사용예시**
  >'top'
  ps
  jovs
  kill
